# 사업 시스템 정리 — "되는 사업을 해라. 우리가 다 검증해드린다."

*작업용 정리 문서. 2026-06-09 피벗 반영판. 빌드 매출 포기, 검증 전문화.*

---

## 1. 한 줄 정의

> **"되는 사업을 해라. 우리가 다 검증해드립니다."**
> AI 시대, 사업 시작 전에 *진짜 시장 수요 + 객단가 + 손익 구조*까지 Mock 사이트 + 실제 광고 + SNS 운영 + 데이터 분석으로 검증해주는 사업 아이디어 검증 전문 서비스.

핵심: 우리는 **사업 빌드를 팔지 않는다**. 검증만 판다. *되는지 안 되는지* 답하는 게 본업.

---

## 2. 왜 빌드를 포기했나 (핵심 결정)

| 빌드 (포기) | 검증 (전문화) |
|---|---|
| 시간 ceiling · 인건비 노예 | 반복 · 확장 가능 |
| AI 코딩으로 단가 박살 | AI 못 함 → 단가 유지 |
| 한국 시장 락인 | 해외 진출 가능 (언어만 변경) |
| 한 고객 1회 결제 | 시리얼 창업자 반복 결제 가능 |

→ 빌드 매출 포기해도 *전문화로 단가 + 확장 + 해외* 다 가져옴.

---

## 3. 핵심 차별점 (방어 가능한 해자 6개)

**한 줄 핵심: 보고서만 주는 시장조사 회사가 아니라, 진짜 시장에 띄워서 답을 가져온다.**

1. **AI는 의견, 우리는 진짜 광고 데이터** — AI는 "좋다"고 한다. 우리는 모르는 사람에게 광고 보여주고 *실제 행동* 측정.
2. **Mock 사이트까지 만든다** — 광고용 진짜 같은 랜딩. 광고만 돌리는 게 아님. 결제 의향 측정까지.
3. **SNS 계정 + 광고 운영까지** — 인스타·구글애즈·페북애즈. 일반인 절대 못 함.
4. **돈 벌리는지까지** — 수요만 보는 게 아니라 객단가 · CAC · LTV · 라이프사이클. 수요 있어도 단가 안 맞으면 망함 (MBTI 케이스).
5. **창업자 실패담** — 본인 AI 믿고 사업 2번 망함 (AI퍼니쳐스, MBTI). "내가 당해봐서 안다" 신뢰.
6. **학벌 + 투명성** — 중앙대 산업보안학과(본인) + 광고홍보학과(파트너). Mock 사이트·광고계정·데이터 dashboard 다 공개. 블랙박스 X.

**왜 "AI로 검증한다"는 차별점이 아닌가:** AI는 시장조사처럼 *쓰여진 글*은 잘 만든다. 근데 진짜 시장 반응은 *모르는 사람의 실제 행동*. 그건 광고 + Mock + 사용자 행동 측정으로만 잡힌다. → AI 보완재 포지션.

---

## 4. 타겟 고객

**1순위 (지금):**
- SaaS · 앱 · 웹서비스 시작하려는 사람
- 이미 아이디어 있고 *지금 시작할 준비* 된 사람
- 사업 처음 or 시리얼 창업자 (재구매 가능성)

**2순위 (확장):**
- 마켓플레이스, B2B SaaS, 소비자 서비스 등 표준 검증 가능 카테고리
- 영문 사이트 출시 후 미국 시장 → 글로벌 시리얼 창업자

**제외:**
- "아이디어 없는데 관심만 있는" 드리머 (자가선별 카피로 차단)
- 오프라인 사업 (광고 검증 한계)
- "은행 만들어주세요" 류 규제 산업

---

## 5. 무엇을 하나 (실제 Deliverable)

**Quick 검증 (50만 / 7일):**
1. Mock 랜딩 페이지 1장 제작
2. 구글애즈 1주 운영 (광고비 5만 포함)
3. GA 이벤트 설정 + 폼 수집
4. 수요 · 클릭 · 폼 데이터 정성 분석
5. Go / No-Go 직관 판단 + 1페이지 리포트
6. 30분 결과 미팅
- 신호 못 잡으면 50% 환불

**Deep 검증 (130만 / 14일):**
- Quick 전체 +
1. 인스타 · SNS 계정 운영 (콘텐츠 5~7개)
2. Facebook · Google 양쪽 광고 (광고비 20만 포함)
3. 사용자 설문 + 인터뷰 (5~10명)
4. CAC · LTV · 객단가 · 손익 시뮬레이션
5. 라이프사이클 분석
6. 객관 리포트 + 다음 액션 권고
- No-Go 시 50% 환불

**검증 통과 후 (옵션):** 외부 빌드 파트너 연결 (수수료 10~20% 검토).

---

## 6. 가격 모델

**원칙: 일하는 양이 아니라 *되는 사업을 안 망치는 가치*로 매긴다.**
시장 기준점: 단순 시장조사 리포트 50~200만. 우리는 *진짜 광고 데이터*까지. 더 비싸지만 더 정확.

**Quick 50만:** 진입 가격. 1인 창업자가 망설이지 않고 결제할 만한 선. *너무 싸면* 의심 사고, *너무 비싸면* 진입장벽.
**Deep 130만:** 정밀 분석 필요한 시점 (본격 자금 투입 전). 정량 데이터 결정자용.

**케이스 5~10개 만든 후 인상 가능:**
- Quick 50 → 70만
- Deep 130 → 180~200만

**리스크 리버설:** Quick 신호 없음 → 50% 환불. Deep No-Go → 50% 환불. 광고비는 환불 X (실제 집행).

## 7. 마케팅 / 메시지

**광고 안 막히는 포지셔닝** (소득 보장 X):
- 페인 훅: "AI한테 시장조사 시켜본 적 있죠?"
- 차별 훅: "사업 시작 전, 검증부터 제대로."
- 결과 훅: "되는 사업을 해라. 우리가 다 검증해드립니다."

**핵심 후크 (롱폼):**
> "AI 믿고 시장조사 시키고, 좋다고 해서 시작했어요. 사업 2개 말아먹었습니다. 사업 시작 전에 *진짜 검증*부터 하세요 — 의견 말고 데이터로."

→ 본인 실패담 (AI퍼니쳐스 v1·v2, MBTI) → 서비스 자연 연결.

**셔츠 비주얼 후크:** "사업 시작 전, 검증부터" — 본인 직접 입고 영상.

**창업자 스토리:**
- 중앙대 산업보안학과
- AI퍼니쳐스 → 부동산 → 공인중개사용 → 다 망함
- MBTI 결제 → 수요는 있었으나 객단가 안 맞아 망함
- 득템잡이 → 처음으로 광고 먼저, 성공
- 결론: "이걸 시스템화해서 다른 사람한테 제공한다"

**금지:** "월 ○○ 보장 / 누구나 부자" 절대 금지. 매출 보장 X.

## 8. 팀 (신뢰 자산)

| 역할 | 학벌 | 담당 |
|---|---|---|
| 본인 (Founder) | 중앙대 산업보안학과 | Mock 사이트 빌드 · 데이터 분석 · 사업 전략 |
| 파트너 | 중앙대 광고홍보학과 | 광고 운영 · SNS 마케팅 · 객단가 분석 |

학벌·역할 공개 = 신뢰 가속. 익명/블랙박스 외주사와 명확한 대비.

---

## 9. 가드레일

- **스코프:** SaaS · 앱 · 웹서비스 표준 카테고리만. 오프라인·규제산업 거절 or 별도 견적.
- **법무:** 매출 보장 광고 절대 금지. Mock 사이트는 *윤리적 페이크도어* (사전등록 안내까지). 개인정보처리방침 · 이용약관 세팅.
- **계약:** "검증 결과" 보장 X, "검증 산출물" 명시 (Mock 사이트, 광고 운영 일수, 리포트 내용 N개). Go/No-Go 합격선 사전 합의.
- **Mock 사이트 처리:** 검증 종료 후 *고객에게 자산 이관* (도메인/디자인). 우리는 권리 없음.

## 10. Risk & 대응

| 리스크 | 대응 |
|---|---|
| 단가 천장 (50~130). 재구매율 낮음. | 시리얼 창업자 타겟. 빌드 외주 수수료. "월 1회 검증 구독" 옵션 검토. |
| "하루 만에 mock"은 마케팅 약속, 실제 2~3일. | 동시 케이스 5~7건/월 제한. 카테고리별 템플릿 + 커스텀 조합으로 효율화. |
| 검증 신호 애매한 회색지대. | 합격선 사전 합의. Deep으로 업셀 통로. |
| 학벌·실명 노출 부담. | 본인 결정. 익명 시 신뢰 부족 — 학벌만이라도 공개 권장. |
| 초기 케이스 없음. | 첫 1번은 *본인 다음 아이디어*로 자기 케이스 만들기 (W1-2). 영업 증거. |

---

## 11. 실행 로드맵 (첫 60일)

- **W1–2:** 본인 다음 사업 아이디어 1개 정해서 우리 프로세스로 Quick 검증 진행 (자기 사례 = 영업 증거). 동시에 랜딩 v2 완성.
- **W2–3:** 카카오톡 채널 개설 + 30분 무료 상담 슬롯 (주 3회). 폼 + 카톡 둘 다 받음.
- **W3–4:** 광고 시작 (구글애즈, 5~10만/일) — *우리 자신* 의 서비스 수요 테스트. 상담 신청 받음.
- **W4–6:** 첫 파운딩 고객 3~5명 모집 (Quick 검증). 진행 + 케이스 스터디 확보.
- **W6–8:** 케이스 후기 받고 가격 인상 검토 (Quick 70, Deep 180). 본격 광고.
- **W8–12:** 영문 사이트 초안 + 미국 진출 사전 조사.

---

## 12. 해외 진출 야망 (3~6개월)

**한국 → 영문 → 미국 시리얼 창업자 시장.**
- 검증 서비스는 *언어만 바꾸면 글로벌*
- 미국 시리얼 창업자는 검증에 *훨씬 더 많이* 결제 (Quick $1,500~3,000 가능)
- 한국 케이스 5~10개 만든 후 진출. 초기 직진 X.
- 영문 광고 후크: "Validate before you build."
